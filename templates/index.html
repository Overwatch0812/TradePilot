TradePilot

Endpoints To use TradePilot database model

For Vendors

  • /api/vendors/: used to display all the vendors in tha database
  • /api/vendors/create/ : used to create new vendor
  • /api/vendors/vendor_id/detail/ : used to view the details of a particular vendor
  • /api/vendors/vendor_id/update/ : sed to update the details of a vendor
  • /api/vendors/vendor_id/delete/ : used to delete a vendor

For Purchase Orders

  • /api/purchase_order/vendor_id/: used to display all the purchase order of a particular vendor in tha database
  • /api/purchase_order/create/ : used to create new Purchase order
  • /api/purchase_order/PO_id/detail/ : used to view the details of a particular Purchase order
  • /api/purchase_order/PO_id/update/ : used to update the details of a Purchase order
  • /api/purchase_order/PO_id/delete/ : used to delete a Purchase order

For Performance Measurement

  • /api/vendors/vendor_id/performance/: retrieves the calculated performance metrics for a specific vendor

For PO Acknowledgement

  • /api/purchase_order//acknowledge/: an endpoint for vendors to acknowledge Purchase order
